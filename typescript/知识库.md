## 一、基础类型
### 1.1 Boolean类型
```ts
let cml:boolean = true
// es5: let cml = true
```
### 1.2 Number类型
```ts
let cml:number = 1
// es5: let cml = 1
```
### 1.3 String类型
```ts
let cml:string = '1'
// es5: let cml = '1'
```
### 1.4 Array类型
```ts
let cml:number[] = [1, 2, 3]
// es5: let cml = [1, 2, 3]

let cml:Array<number> = [1, 2, 3]
// es5: let cml = [1, 2, 3]
```

### 1.5 enum枚举类型
#### 1.5.1 数字的枚举
```ts
enum Direction {
  NORTH, // 默认0
  SOUTH, // 默认1
  EAST, // 默认2
  WEST, // 默认3
}
let dir: Direction = Direction.EAST;
console.log('dir', dir) // 2
```
默认为0,1,2,3...，如果自定义数字，则后面的枚举在前一个枚举值上递增

```ts
enum Direction1 {
  NORTH = -1,
  SOUTH, // 变成0
  EAST = 4,
  WEST, // 变成5
}
```
